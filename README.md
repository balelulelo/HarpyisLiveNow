## Harpy is Live !: Multiplayer Live Stream Simulator over TCP

A TCP based live stream simulator where multiple viewers connect to watch Harpy, a chaotic energetic streamer character. Viewers can chat, send gifts, donate, subscribe, and like. Harpy will respond in real time with her own personality. She also broadcasts live commentary on her own, making the stream feel alive even when nobody is chatting.

## Requirements

* Python 3.10 or higher
* No external dependencies for the current version (all standard library)
